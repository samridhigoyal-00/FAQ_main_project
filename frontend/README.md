# SupportHub Frontend

This is the React frontend for the FAQ App. It is built using **Vite**, **React 18**, and **React Router v6**.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. The application will be available at [http://localhost:3000](http://localhost:3000).

> **Note:** For full setup instructions, environment variables, and backend configuration, please see the [Main README](../README.md) in the root directory.
